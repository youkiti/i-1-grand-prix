# 目的
- 「新しい論点抽出レポート」の内容を「現在のQ&Aリスト」に**統合**してください。
- **分析の主眼**: **「{{ focus }}」**に関連する内容に限定してください。会議の運営方法などの事務的な詳細は**除外**してください。

# 統合ルール（重要）
1. **重複の統合**: 同じ論点が複数のドキュメントで言及されている場合、**1つの項目にまとめて**ください。
2. **引用の統合**: 同じ論点に複数の出典がある場合、`[出典: file1.pdf, p.1; file2.pdf, p.5]` のようにセミコロンで区切って併記してください。
3. **簡潔化**: 冗長な記述は要点を抽出して簡潔にまとめてください。ただし、**重要な論点や対立軸は省略しない**でください。
4. **引用の維持**: 出典タグ `[出典: ファイル名, p.X]` は必ず維持してください。
5. **対立軸の統合**: 同じ論点に関する対立軸が複数のドキュメントで言及されている場合、以下のルールで統合してください：
   - 同一の対立軸（例: 甲案 vs 乙案）は1つにまとめ、出典を併記
   - 新たな立場や根拠が追加された場合は、表に行を追加
   - 合意形成の状況が進展した場合（例: 継続検討→決着）は、最新の状況に更新し、経緯を補足

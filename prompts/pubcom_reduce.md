# 目的
- 「新しい個別分析結果」の内容を「現在のレポート」に**統合**してください。
- **文体**: 単なる箇条書きの羅列や体言止めは避け、**完全な文章（～です。～ます。）で、文脈や背景がわかるように**記述してください。

# 統合ルール（重要）
1. **重複の統合**: 同じ論点が複数のコメントで言及されている場合、**1つの項目にまとめて**ください。
2. **引用の統合**: 同じ論点に複数のコメントIDがある場合、`[パブコメ: ID1; ID2]` のようにセミコロンで区切って併記してください。
3. **簡潔化**: 冗長な記述は要点を抽出して簡潔にまとめてください。ただし、**重要な論点や対立軸は省略しない**でください。
4. **引用の維持**: 出典タグ `[パブコメ: コメントID]` は必ず維持してください。
